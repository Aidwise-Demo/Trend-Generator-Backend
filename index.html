Endpoints

Endpoint Description
POST /limited_generate Endpoint for limited generation
GET /inputlist Endpoint for getting list of all Requests
GET /status/{folder_name} Endpoint for Status Checking
GET /data/{folder_name} Endpoint for getting output in JSON
GET /DownloadFile/{folder_name} Endpoint for Downloading files

Example JSON Input for Postman - Limited-Generation

{
    "Trends": "Social",
    "filename": "Output",
    "no_of_gen": 2,
    "email": "sidhantraj0110@gmail.com",
    "dept": "Law enforcement",
    "additional_details": "",
    "geography": "Australia"
}

Note: You will get the folder name as an output from the limited_generation endpoint.
